=== CIVIL ENGINEERING CPM — THE STANDARD ===

In civil engineering, the Critical Path Method (CPM) uses:

FORWARD PASS (Q→A direction):
- Start from the beginning
- Calculate Earliest Start (ES) and Earliest Finish (EF) for every activity
- EF = ES + Duration
- The forward pass answers: "What is the soonest everything can be done?"

BACKWARD PASS (A→Q direction):
- Start from the end (the required completion date)
- Calculate Latest Start (LS) and Latest Finish (LF) for every activity
- LS = LF - Duration
- The backward pass answers: "What is the latest everything can start without delaying the project?"

FLOAT = LS - ES (or LF - EF)
- Zero float = CRITICAL PATH. Any delay here delays everything.
- Positive float = buffer. Some slack.

Engineers do this 3 TIMES:
- Pass 1: Initial calculation
- Pass 2: Verification (independent recalculation)
- Pass 3: Confirmation (third party or senior engineer checks)

The purpose is CERTAINTY. Not speed. CERTAINTY. Because people die if the critical path is wrong. A bridge collapses. A tunnel floods. A building falls.
